# Projet : Data cleaning d'une base de données 

Au cours de ce projet, j'ai fait le traitement des données sur un dataset. Il comprenait 10100 lignes.

# Etapes réalisées 
1. suppression des doublons
2. suppression des valeurs abbérantes(outliers)
3. imputation des valeurs manquantes avec KNNImputer du  module sklearn.impute

# Résultat final
Le dataset final comprend 100000 lignes et est prête pour une analyse avancée

# Technologies utilisées

1. Python
2. Pandas
3. Scikit-learn
4. Jupyter Notebook
